Detailed Description

Factory methods to create a protocol stack.

To create an initiator, use one of the following functions:

  • ssp21::initiator::factory::shared_secret_mode()
  • ssp21::initiator::factory::qkd_mode()
  • ssp21::initiator::factory::preshared_public_key_mode()
  • ssp21::initiator::factory::certificate_public_key_mode()

To create a responder, use one of the following functions:

  • ssp21::responder::factory::shared_secret_mode()
  • ssp21::responder::factory::preshared_public_key_mode()
  • ssp21::responder::factory::qkd_mode()
  • ssp21::responder::factory::certificate_public_key_mode()
